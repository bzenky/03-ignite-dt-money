## Sobre
Este é o terceiro projeto da atualização 2022 da trilha ReactJS do Ignite.

## Rotina
- Filtrar as transações
- Inserir uma nova transação

## Stacks
- ReactJs
- ViteJs
- Typescript

## Rodando

1. Clonar o repositório, instalar as dependências
```
yarn / npm install
```

2. Subir o json-server
```
yarn dev:server
```

3. Subir o projeto
```
yarn dev
```
